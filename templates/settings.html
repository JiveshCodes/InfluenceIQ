API access requires a Pro or Agency plan. Keep this key secret — do not share it publicly.

Example Request

POST /api/predict
Content-Type: application/json
Authorization: Bearer YOUR_API_KEY

{
  "category": "Tech",
  "subcategory": "AI/ML",
  "platform": "YouTube",
  "budget": 20000
}
