The Technology Stack

Built on modern ML infrastructure — not keyword matching or manual curation.

🤗
Sentence Transformers
all-MiniLM-L6-v2 model encodes influencer profiles and brand briefs into 384-dimension semantic vectors for deep contextual matching.
NLP · Transformers · BERT
📐
Cosine Similarity
Query vectors are matched against pre-encoded influencer embeddings using cosine similarity — capturing semantic meaning beyond keyword overlap.
Linear Algebra · Embeddings
⚖️
Weighted ML Scoring
Five-factor composite score: semantic similarity (30%), engagement (25%), category match (20%), keyword alignment (15%), fraud safety (10%).
Machine Learning · Scoring
🛡️
Fraud Detection
Each influencer carries an audience authenticity score derived from follower-to-engagement ratios and growth pattern analysis.
Anomaly Detection · NLP
💬
Explainable AI
Every recommendation is accompanied by clear human-readable reasons. No black box scores — full interpretability for marketing teams.
XAI · Interpretability
🐍
Python + Flask
Backend built on Flask with pandas, scikit-learn, and sentence-transformers. Embeddings are pre-computed at startup for fast real-time inference.
Flask · scikit-learn · pandas

Dataset Quality

Our influencer dataset is hand-curated and regularly verified. Every entry includes:

  • Verified real follower counts from public profiles
  • Accurate engagement rate calculations
  • Correct fraud risk labels (no real influencers marked fake)
  • Multi-platform coverage: Instagram, YouTube, Twitter, LinkedIn
  • Geographic diversity: India, USA, UK, UAE, and global creators
  • Platform verification status where available
